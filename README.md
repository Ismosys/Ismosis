# Ismosis

Patent illustration studio website. Built with Next.js 14 App Router, Tailwind CSS, and Framer Motion.

## Local development

```bash
npm install
npm run dev
```

Then open http://localhost:3000

## Production build

```bash
npm run build
npm run start
```

## Project structure

- `app/` page routes and layout
- `components/` shared components
- `components/sections/` homepage sections
- `components/illustrations/` SVG patent illustrations
- `lib/` shared utilities

## Pages

- `/` Home
- `/services` Services detail
- `/portfolio` Portfolio with category filtering
- `/process` Workflow
- `/pricing` Pricing tiers
- `/about` About the studio
- `/contact` Quote request form
